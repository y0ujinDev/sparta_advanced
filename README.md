# 스파르타 심화 주차 개인 과제

# 환경변수

- .env.sample 파일 참고

# 실행 방법

```bash
yarn install
yarn dev
yarn test
```

# 과제 설명

- **3-Layered Architecture** 이해하고 적용하기
- 객체 지향 프로그래밍의 기초인 **Class** 이해하고 적용하기
- **Prisma** 사용해보기

# 기술 스택

- JavaScript(ES6+), Node.js, Express, Prisma, AWS EC2, AWS RDS

# 개발 도구

- Yarn, Nodemon, Jest, Cross-env

# 주요 기능

- 사용자 관련 기능
  - 회원가입
  - 로그인
  - 유저 정보 확인
- 상품 관련 기능
  - 전체 상품 목록 조회
  - 개별 상품 상세 조회
  - 상품 등록
  - 상품 수정
  - 상품 삭제
